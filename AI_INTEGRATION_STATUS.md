## Проблемы с текущей конфигурацией

⚠️ **Ошибки подключения к Hugging Face** - в логах сервера видны ошибки "No Inference Provider available for model"

Это означает, что:
1. API ключ настроен правильно (начинается с `hf_`)
2. Но выбранные модели недоступны для текущего аккаунта Hugging Face

## Решения

### 1. Активируйте модели в Hugging Face:
- Перейдите на страницу модели [google/flan-t5-large](https://huggingface.co/google/flan-t5-large)
- Нажмите кнопку "Use this model"
- Подтвердите использование модели в своих проектах
- Повторите для других моделей из списка

### 2. Проверьте квоты:
- Бесплатные аккаунты Hugging Face имеют ограничения
- Рассмотрите возможность обновления до Pro плана

### 3. Используйте тестирование:
- В интерфейсе генератора нажмите "Тест подключения"
- Это поможет определить точную причину проблемы
