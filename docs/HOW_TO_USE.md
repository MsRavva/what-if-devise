# Инструкция по использованию приложения "What If Devise"

## Общее описание

"What If Devise" - это веб-приложение, которое позволяет создавать альтернативные сценарии для ваших историй с помощью искусственного интеллекта. Вы можете генерировать различные варианты развития событий, исследовать "что если" вопросы и сохранять свои сценарии для дальнейшего использования.

## Начало работы

1. **Регистрация/Вход в систему**
   - Нажмите на кнопку "Войти" на главной странице
   - Вы можете зарегистрироваться с помощью email и пароля или использовать вход через Google
   - После входа вы получите доступ ко всем функциям приложения

2. **Создание альтернативного сценария**
   - Нажмите на кнопку "Что было бы, если..." на главной странице (функция пока в разработке)
   - В будущем здесь будет возможность ввести вашу историю и задать вопрос "что если"

3. **Использование шаблонов**
   - Перейдите на страницу "Шаблоны"
   - Выберите один из 8 готовых шаблонов для вдохновения
   - Используйте шаблон как основу для своего сценария

4. **Управление профилем**
   - Перейдите на страницу "Профиль" для управления вашей учетной записью
   - Здесь вы можете изменить настройки и посмотреть вашу статистику

5. **Просмотр истории**
   - На странице "История" вы можете посмотреть все созданные вами сценарии
   - Здесь же можно редактировать или удалять сохраненные работы

## Функции приложения

- **Генерация сценариев с помощью ИИ**: Создавайте уникальные альтернативные варианты развития событий
- **Сохранение и организация**: Сохраняйте ваши сценарии и организуйте их в удобной форме
- **Шаблоны**: Используйте готовые шаблоны для быстрого старта
- **Аутентификация**: Безопасный вход и регистрация с поддержкой Google OAuth
- **Персонализация**: Настройте приложение под свои предпочтения

## Вход через Google

Приложение поддерживает вход через Google. Для этого:
1. Нажмите на кнопку "Войти" на главной странице
2. Выберите опцию входа через Google
3. Подтвердите аутентификацию в открывшемся окне Google

## Безопасность

- Все данные пользователей защищены с помощью современных методов шифрования
- Приложение использует безопасные методы аутентификации
- Ваши персональные данные не передаются третьим лицам

## Техническая поддержка

Если у вас возникли проблемы с использованием приложения:
1. Проверьте соединение с интернетом
2. Обновите страницу
3. Очистите кэш браузера
4. Обратитесь в техническую поддержку через страницу профиля

## Часто задаваемые вопросы

**Q: Могу ли я использовать приложение бесплатно?**
A: Да, приложение полностью бесплатно для всех пользователей.

**Q: Сколько сценариев я могу создать?**
A: Ограничений по количеству создаваемых сценариев нет.

**Q: Могу ли я делиться своими сценариями с другими?**
A: В настоящее время функция публикации сценариев находится в разработке.
