# Как использовать генератор сценариев "Что если"

## Шаг 1: Настройка Hugging Face API ключа

1. Зарегистрируйтесь или войдите в аккаунт на [Hugging Face](https://huggingface.co)
2. Перейдите в [настройки токенов](https://huggingface.co/settings/tokens)
3. Создайте новый токен с ролью 'Read'
4. Скопируйте токен (он должен начинаться с `hf_`)
5. Создайте файл `.env.local` в корне проекта:
   ```bash
   cp .env.local.example .env.local
   ```
6. Откройте файл `.env.local` и замените `hf_your_actual_api_key_here` на ваш реальный API ключ

## Шаг 2: Запуск приложения

1. Установите зависимости:
   ```bash
   pnpm install
   ```

2. Запустите сервер разработки:
   ```bash
   pnpm dev
   ```

3. Откройте приложение в браузере: http://localhost:3000

## Шаг 3: Создание альтернативного сценария

1. Перейдите на страницу "Генератор сценариев"
2. Введите свою историю в поле "Ваша история"
3. Сформулируйте вопрос "Что было бы, если..." в соответствующем поле
4. Нажмите кнопку "Создать альтернативный сценарий"
5. Дождитесь генерации сценария (может занять 10-30 секунд)

## Советы для лучшего результата

### Для истории:
- Используйте подробные описания событий
- Включите эмоции и чувства персонажей
- Добавьте контекст и детали обстановки
- Минимум 10 символов

### Для вопроса "Что если":
- Формулируйте четко и конкретно
- Используйте открытые вопросы (что было бы, если бы...)
- Избегайте слишком абстрактных формулировок
- Минимум 5 символов

### Пример хорошей истории:
```
Вчера я встретил старого друга в маленьком кафе в центре города. Мы не виделись больше пяти лет. Он рассказал мне, что переехал из Москвы в небольшой provincial город и открыл там свою IT-компанию. Я был удивлен его успехом, ведь раньше он всегда жаловался на отсутствие амбиций.
```

### Пример хорошего вопроса:
```
что было бы, если бы я решил присоединиться к его компании в качестве технического директора?
```

## Решение проблем

### Если сценарий не генерируется:
1. Проверьте, что API ключ настроен правильно
2. Нажмите кнопку "Тест AI" для проверки подключения
3. Попробуйте перегенерировать сценарий
4. Измените формулировку вопроса

### Если сценарий некачественный:
1. Добавьте больше деталей в исходную историю
2. Уточните вопрос "Что если"
3. Попробуйте перегенерировать

### Если возникают ошибки:
1. Проверьте подключение к интернету
2. Убедитесь, что API ключ действителен
3. Обратитесь к руководству по устранению неполадок
